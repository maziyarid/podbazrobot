# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-12-28

### Added
- Initial release of PodBaz Robot plugin
- Admin menu and settings page
- Database table for activity logging
- Plugin activation and deactivation hooks
- Settings page with configurable options
- Logs page to view plugin activity
- Admin CSS and JavaScript assets
- Translation support
- Security features including input sanitization and validation
- WordPress coding standards compliance
